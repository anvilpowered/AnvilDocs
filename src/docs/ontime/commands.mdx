## Add

Permission: `ontime.admin.edit`

Aliases: `a, add`

Usage: `ontime add <user> <time>`

## Check

- Permissions

  - User: `ontime.user.check`

  - Admin: `ontime.admin.check`

Aliases: `c, check info, i`

Usage: `ontime check [<user>]`

## Import

About: This commands is used to import data from RankUp. For more information, click [here](https://github.com/AnvilPowered/OnTime/wiki/Importing-Data-From-Rankup)

Usage: `ontime import <path>`

## Set

Permission: `ontime.admin.edit`

Aliases: `set, s, settotal, st`

Usage: `ontime set <user> <time>`

## Set Bonus

Permission: `ontime.admin.edit`

Aliases: `setbonus, sb`

## Version

Aliases: `v, version`
