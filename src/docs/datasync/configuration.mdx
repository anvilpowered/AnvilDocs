---
title: DataSync Configuration
---

## Serialize Section
|Setting|Description|Default|
|-------|-----------|-------|
|deserializeOnJoin|Whether DataSync should deserialize players on join|true|
|deserializeOnJoinDelayMillis|The number of milliseconds DataSync should wait before downloading a snapshot on join|0|
|enabledSerializers|Things to sync to DB| datasync:experience, datasync:gameMode, datasync:health, datasync:hunger, datasync:inventory|
|serializeOnDeath|Whether DataSync should serialize players to the DB on death|true|
|serializeOnDisconnect|Whether DataSync should serialize players to the DB on disconnect|true|

## Snapshot Section
|Setting|Description|Default|
|-------|-----------|-------|
|minCount|Minimum number of snapshots to keep before deleting any|5|
|optimizationStrategy|Snapshot optimization strategy|"60:24", "1440:7"|
|uploadInterval|Interval for automatic serialization task|5|
