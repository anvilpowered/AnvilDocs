---
title: Catalyst Commands
---


## Ban

Permission: `catalyst.command.ban`

Aliases: `cban`

Usage: `/ban <name> [reason]`

Description: Bans a player.


## TempBan

Permission: `catalyst.command.tempban`

Aliases: `ctempban`

Usage: `/tempban <user> <duration> [reason]`

Description: Temporarily bans a player.

## Broadcast

Permission: `catalyst.command.broadcast`

Aliases: `cbroadcast`

Usage: `/broadcast [message]`

Description: Broadcast a message to the proxy.


## Find

Permission: `catalyst.command.find`

Aliases: `cfind`

Usage: `/find <player>`

Description: Find which server a player on.

## Info

Permission: `catalyst.command.info`

Aliases: `cinfo`

Usage: `/info <player>`

Description: Get the information of a player.


## Kick

Permission: `catalyst.command.kick`, `catalyst.command.kick.exempt`

Aliases: `ckick`

Usage: `/kick <player> [reason]`

Descrption: Kick a player.


## Swear / Exception

Permission: `catalyst.command.language.admin`

Usage: `/swear (add|remove <word>)`

Usage: `/exception (add|remove <word>)`

Additional Usage:

* Permission: `catalyst.command.language.list`

* Usage: `/swear list`

* Usage: `/exception list`

Description: Add or remove swear/exception words from the list.

## List

Permission: `catalyst.command.list`

Aliases: `clist`

Usage: `/list`


## Message

Permission: `catalyst.command.message`

Aliases: `msg`, `tell`, `whisper`, `m`, `t`

Usage: `/msg <player> [message]`

Description: Send a message to another player.


## Mute

Permission: `catalyst.command.mute`, `catalyst.command.mute.exempt`

Aliases: `cmute`

Usage: `/mute <player> [reason]`

Description: Mute a player.


## Nickname

Permission: `catalyst.command.nickname`, `catalyst.command.nickname.color`, `catalyst.command.nickname.other`

Aliases: `cnick`, `nickname`

Usage: `/nick <nickname>` or `/nick <player> <nickname>`

Description: Set your nickname or that of another player.


## Send

Permission: `catalyst.command.send`

Aliases: `csend`

Usage: `/send <player> <server>`

Description: Send a player to a server.


## SocialSpy

Permission: `catalyst.command.socialspy`, `catalyst.command.socialspy.onjoin`

Aliases: `ss`

Usage: `/socialspy`

Description: View private messages of other players.

## Stafflist

Permission: `catalyst.stafflist.base`

* Owner: `catalyst.stafflist.owner`

* Admin: `catalyst.stafflist.admin`

* Staff: `catalyst.stafflist.staff`

Usage: `/stafflist`

Description: View all active staff.

## TempMute

Permission: `catalyst.command.mute`

Aliases: `ctempmute`

Usage: `/tempmute <player> <duration> [reason]`

Description: Temporarily mute a player.

## Ignore

Permission: `catalyst.command.ignore`, `catalyst.command.ignore.exempt`

Aliases: `cignore`

Usage: `/ignore <player>`

Description: Ignore another player.
