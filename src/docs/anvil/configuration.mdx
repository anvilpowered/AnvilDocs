## Datastore Section

|Setting|Description|Default|
|-------|-----------|-------|
|dataDirectory|Directory for extra data|anvil|
|dataStoreName|Determines which storage option to use|xodus|
|mondogb.authDb|MongoDB database to use for authentication|admin|
|mongodb.connectionString|Custom MongoDB connection string that will be used instead of the connection credentials|mongodb://admin:password@localhost:27017/anvil?authSource=admin|
|mongodb.dbname|MongoDB database name|anvil|
|mongodb.hostname|MongoDB hostname|localhost|
|mongodb.password|MongoDB password|password|
|mongodb.port|MongoDB port|27017|
|mongodb.useAuth|Whether to use authentication (username/password) for MongoDB connection|false|
|mongodb.useConnectionString|Whether to use the connection string instead of the normal connection info|false|
|mongodb.useSrv|Whether to interpret the MongoDB hostname as a SRV record|false|
|mongodb.username|MongoDB username|admin|

## Server Section

|Setting|Description|Default|
|-------|-----------|-------|
|name|Server name|server|
|proxyMode|Enable this if your server is running behind a proxy|false|
|regeditAllowSensitive|Whether the regedit command should have access to sensitive settings such as connection details|false|
